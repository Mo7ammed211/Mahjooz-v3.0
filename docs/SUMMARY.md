## 🎯 كيفية البدء؟

### الخيار 1: تجربة سريعة (5 دقائق)
```bash
# فقط افتح index.html في المتصفح واختر حساب تجريبي
```

### الخيار 2: تشغيل محلي (15 دقيقة)
