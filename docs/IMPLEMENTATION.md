## 🔧 كيفية الاستخدام

### للعملاء:
1. اختر دخول كعميل
2. تصفح الخدمات
3. احجز خدمة (تحتاج رصيد)
4. اتبع الطلب
5. قيّم الخدمة بعد الانتهاء

### لأصحاب الخدمات:
1. اختر دخول كبائع
2. غيّر كلمة المرور عند أول دخول
3. استقبل الطلبات واقبلها
4. تابع الأرباح
5. اطلب السحب

### للمندوبين:
1. اختر دخول كمندوب
2. استقبل طلبات التوصيل
3. حدّث حالة التوصيل
4. تابع أرباح التوصيل

### للإدارة:
1. اختر دخول كمدير
2. أضف التصنيفات والخدمات والإعلانات
3. أدر الطلبات والمستخدمين
4. وافق على طلبات الشحن
5. انظر التقارير والإحصائيات
