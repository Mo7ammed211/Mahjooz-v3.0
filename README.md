# 🚀 منصة محجوز — Mahjooz Platform

منصة خدمات وحجوزات شاملة تجمع بين الفنادق والخدمات المهنية والمتاجر الطبية في تطبيق واحد.

---

## 🎯 نظرة عامة

**محجوز** هي منصة متعددة الأطراف تربط:
- **العملاء** (يبحثون عن الخدمات)
- **أصحاب الخدمات** (يقدمون الخدمات)
- **مندوبي التوصيل** (يوصلون الطلبات)
- **الموظفين** (يدعمون العملاء)
- **الإدارة** (تدير المنصة)

## 👥 الأدوار والصلاحيات

### 👑 المدير (Admin)
- **الصلاحيات:**
  - إضافة وتعديل وحذف التصنيفات والخدمات
  - إدارة جميع المستخدمين
  - مراجعة وقبول الطلبات
  - إدارة المحافظ والشحن
  - إنشاء الإعلانات المميزة
  - عرض التقارير والإحصائيات

### 🖥️ موظف خدمة العملاء (Staff)
- **الصلاحيات:**
  - إضافة خدمات نيابة عن البائعين
  - مراجعة طلبات الشحن
  - دعم العملاء

### 🏪 صاحب الخدمة (Vendor)
- **الصلاحيات:**
  - عرض الطلبات الواردة
  - قبول أو رفض الطلبات
  - تتبع الأرباح
  - طلب السحب
  - تغيير كلمة المرور عند أول دخول

### 🚗 مندوب التوصيل (Driver)
- **الصلاحيات:**
  - عرض الطلبات المكلف بها
  - تحديث حالة التوصيل
  - عرض عنوان العميل
  - تتبع أرباح التوصيل

### 👤 عميل (Customer)
- **الصلاحيات:**
  - البحث والحجز من الخدمات
  - دفع من المحفظة
  - تتبع الطلبات
  - تقييم الخدمات والمندوبين
  - إدارة محفظتي

## 🛠️ التثبيت والإعداد

### 1. متطلبات النظام
- متصفح حديث
- اتصال إنترنت

### 2. التثبيت المحلي
```bash
# انسخ المشروع
git clone https://github.com/your-repo/mahjooz.git
cd mahjooz

# إذا كان لديك Node.js
npm install

# شغّل server محلي (للاختبار)
npx http-server
```

### 3. إعدادات Firebase
- اذهب إلى [Firebase Console](https://console.firebase.google.com)
- أنشئ مشروع جديد
- فعّل Authentication (Email/Password)
- أنشئ Firestore Database
- انسخ معرّفات المشروع إلى `firebase-config.js`

### 4. النشر على الويب
```bash
firebase init hosting
firebase deploy
```

## 📊 قاعدة البيانات (Firestore)

### Collections

**users** - معلومات المستخدمين
```javascript
{
  name: string,
  email: string,
  role: 'admin'|'staff'|'vendor'|'driver'|'customer',
  phone: string,
  createdAt: timestamp
}
```

**orders** - الطلبات
```javascript
{
  orderId: string (MJZ-2026-000001),
  customerId: string,
  vendorId: string,
  driverId: string,
  svcId: string,
  total: number,
  status: 'pending'|'accepted'|'with_driver'|'completed'|'cancelled',
  createdAt: timestamp
}
```

**wallets** - المحافظ
```javascript
{
  balance: number,
  uid: string,
  createdAt: timestamp
}
```

**ratings** - التقييمات
```javascript
{
  orderId: string,
  customerId: string,
  vendorStars: number (1-5),
  driverStars: number (1-5),
  vendorComment: string,
  driverComment: string,
  createdAt: timestamp
}
```
